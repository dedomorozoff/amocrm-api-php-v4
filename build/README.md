# Инструменты сборки

Эта папка содержит инструменты и документацию для сборки и публикации релизов.

## Структура

- `build-dev-release.sh` - Скрипт сборки для Linux/Mac/Git Bash
- `build-dev-release.bat` - Скрипт сборки для Windows
- `BUILD.md` - Инструкция по локальной сборке релизов
- `PUBLISH.md` - Инструкция по публикации на Packagist.org
- `releases/` - Директория с собранными релизами (создается автоматически, игнорируется git)

## Использование

### Сборка локального релиза

**Linux/Mac/Git Bash:**
```bash
./build/build-dev-release.sh [версия]
```

**Windows:**
```cmd
build\build-dev-release.bat [версия]
```

### Публикация на Packagist

См. [PUBLISH.md](PUBLISH.md) для подробных инструкций.

## Примечания

- Скрипты автоматически определяют корневую директорию проекта
- Собранные релизы сохраняются в `build/releases/`
- Архивы создаются в формате ZIP
